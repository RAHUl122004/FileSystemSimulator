# FileSystemSimulator
A simple file system simulator in C++ using vectors and structs
A simple file system simulator in C++ that allows creating, reading, writing, and listing files using data structures like vectors and structs.
## Features
- Create new files
- Write data to files
- Read data from files
- List all files in the system
- Simple console-based menu
## How to Run
1. Clone the repository:
   git clone https://github.com/RAHUl122004/FileSystemSimulator.git 
2. Navigate to the project folder:
   cd FileSystemSimulator
3. Compile the code:
   g++ main.cpp -o FileSystemSimulator.exe
4. Run the executable:
   .\FileSystemSimulator.exe
